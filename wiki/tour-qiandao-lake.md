## 路线

杭州客运中心（8:30）-> 千岛湖客运西站 -> 旅馆 -> 千岛湖景区综合管理处 -> 龙珠广场 
-> 观松台 -> 渔乐岛 -> 月光岛 -> 千岛湖景区综合管理处 -> 千岛湖客运西站 -> 回城
