# 越南芽庄

## 地图
无

## 路线
坐灰机->坐车->坐灰机

## 足迹

路线：跟着导游走，没啥好说的

```
1.坐灰机（21点半）去金兰国际机场;
2.入住酒店（Cam Hải Đông);
2.芽庄大教堂->占婆塔->酒店;
3.自由行;
4.买翡翠，买橡胶制品，回去;
```
